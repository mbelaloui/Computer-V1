# Projecti title Computer V1

## Description

The aim of this project is to do a program for solving polynomial equation of degree less than or equal to 2. 

The main part of the project is the passing part which is more flexible than the subject is demanding


### Basic format expression managed
```
	[+-]C * X^0 [+-]b * X^1 [-+]A * X^2 = 0 / A,B,C ∈ ℝ
```

### Other format expression managed
```
	A, X, AX, A*X, A*Xe, A*X^e
	[+-]C * X^Z [+-]b * X^Y [-+]A * X^P [= 0 optional] / Z,Y,P ∈ ℕ 
	[+-]C * X^0 [+-]b * X^1 [-+]A * X^2 [= DX^e optional]

```
